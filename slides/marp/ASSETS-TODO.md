# ASSETS-TODO: screenshots to capture for the Course 2 deck

The deck builds clean today because every web asset it cannot fetch is stood in
by a placeholder frame in `slides/figures/` (dark dashed card + URL + capture
instruction, generated by `slides/figures/generate-web-placeholders.py`).

**To swap one in:** capture the screenshot per the instruction below, save it
over the same filename in `slides/figures/`, rebuild. No markdown edits needed.

```bash
cd slides/marp && pnpm exec marp --config marp.config.js --allow-local-files \
  deck/course2.md -o out/course2.html
```
